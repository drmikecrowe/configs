# Prettier config

## Usage

### Install

```bash
$ npm i @drmikecrowe/config-npm-check-updates -D
# OR
$ yarn add @drmikecrowe/config-npm-check-updates -D
```

### Extend this config

in `package.json`

```json
{
    "ncurc": "@drmikecrowe/config-npm-check-updates"
}
```
